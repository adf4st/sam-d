JWT

JSON Web Tokens are generated by the API and issued to authenticated web clients. This provide a self-contained and stateless form of authorization. The angular-jwt library is used on the front-end to provide common JWT related functions.

See the Code »

User Accounts

The SAM-D Sample App comes with support for authenticated user accounts. The API layer persists account data in a MongoDb instance, supports multiple permission levels, and makes it easy to restrict access to resources.

Try Logging In »

Fully Containerized

The SAM-D Sample App comes neatly packaged in three Docker containers to ease the deployment process. Launch the entire demo app in a single command by using the included docker-compose file. See the GitHub repo for more.

Login and Register

Create a new account or log into an existing account on the user accounts page. Users authenticate with a username and password and JSON Web Tokens are used for web client authorization to the API.

Alerts

The included alerts service lets you show a dismissible alert from any page in the application. Just include 'AlertsService' in your controller and call the '.addAlert()' function!
